# Contributing

Thank you for your interest in contributing to this project.

## What this repository contains

This repository currently hosts a React + Vite frontend scaffold. The backend, API integration, database, machine learning, and tests are not implemented yet.

## How to contribute

- Open issues for missing functionality or bugs.
- Submit PRs to add missing backend/API features, tests, or documentation.
- Keep changes focused and avoid redesigning the entire structure.

## Code style

- Use the existing React and Vite setup.
- Keep formatting consistent with the current codebase.
- Run `npm run lint` before submitting changes.

## Local development

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the dev server:
   ```bash
   npm run dev
   ```
3. Verify production build:
   ```bash
   npm run build
   ```
4. Run lint checks:
   ```bash
   npm run lint
   ```

## Notes

This project is a prototype and should be extended gradually, beginning with a backend API layer and test coverage.
